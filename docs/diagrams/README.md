# Diagrams

- **Mermaid** diagrams live inline in module/project READMEs (render natively on GitHub, easy to diff).
- **`.drawio`** sources for polished architecture diagrams live here, each with a PNG/SVG export alongside for embedding.

As modules and projects are built to depth, their `.drawio` sources land here named by area (e.g. `project-07-platform.drawio`, `03-ingestion.drawio`). See [`../aws-icons/`](../aws-icons/) for how to use official AWS icons in draw.io.
