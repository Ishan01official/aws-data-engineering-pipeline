# Diagrams

Source files for the architecture diagrams used across the modules.

- Inline diagrams are authored in **Mermaid** directly inside module READMEs (they render natively on GitHub).
- Diagrams complex enough to need a real canvas are kept here as **`.drawio`** files (open at [app.diagrams.net](https://app.diagrams.net)) with a PNG/SVG export alongside for inline embedding.

As modules are built to full depth, their `.drawio` sources land here, named by module (e.g. `03-ingestion-pipeline.drawio`).
